Wall Attention is the Gibbs Kernel

June 2026

Softmax attention is akin to kernel regression with the Gaussian Kernel and Nadaraya-Watson estimator. Wall Attention, which generalizes diagonal forget gates to attention, lets length scale move per channel and per position. Wall Attention is akin to the Gibbs kernel from kernel regression theory, which is the classical non-stationary generalization of the RBF.

The perspective that attention is kernel regression is well known at this point so I won't re-hash it fully here. Softmax attention is the Nadaraya-Watson estimator, a kernel-weighted average of values, and the test-time regression line of work makes this precise [2][3][4][5]. The kernel is $\kappa(q,k) = \exp(q^\top k)$. Under QK-norm, $\lVert q\rVert = \lVert k\rVert = 1$, so $q^\top k = 1 - \tfrac12\lVert q - k\rVert^2$ and the kernel is a Gaussian with a single unit length scale.

$$\kappa(q_i, k_j) = e \cdot \exp\!\left(-\frac{\lVert q_i - k_j\rVert^2}{2\,\ell^2}\right), \qquad \ell = 1$$

The common constant $e$ cancels in the softmax denominator. The observation to make is that $\ell$ is fixed at $1$ and identical everywhere, which makes the kernel stationary - it only sees the distance $\lVert q - k\rVert$, and it treats every region of representation space and every position in the sequence exactly the same. My previous work on Wall Attention [1] generalizes softmax attention to let $\ell$ move, and the resulting object has a classical analogy in kernel regression theory.

The Gibbs kernel

A stationary kernel assumes the function it models is equally smooth everywhere. For sequences in online regression, this is the wrong prior due to nonuniform information density. The classical fix from the Gaussian process literature is to give every input its own length scale. The Gibbs kernel [6] is the canonical non-stationary RBF kernel with a per-coordinate length scale $\ell_n(x)$.

$$k_{\mathrm{Gibbs}}(x, x') = \prod_{n=1}^{d} \left(\frac{2\,\ell_n(x)\,\ell_n(x')}{\ell_n(x)^2 + \ell_n(x')^2}\right)^{\!1/2} \exp\!\left(-\sum_{n=1}^{d} \frac{(x_n - x'_n)^2}{\ell_n(x)^2 + \ell_n(x')^2}\right)$$

The exponent is a squared distance where each coordinate is divided by a position-dependent length scale, and the prefactor in front keeps the kernel positive-definite as that length scale changes from point to point. If we set $\ell_n(x) \equiv \ell$ for all $n$ and $x$, the prefactor collapses to $1$ and the exponent becomes $-\lVert x - x'\rVert^2/(2\ell^2)$ which recovers the stationary RBF. Intuitively, the Gibbs kernel is the minimal generalization of the RBF that lets the length scale vary. We will show the Gibbs kernel lives inside Wall Attention.

The Gibbs kernel anchored at $x_0$, against the stationary RBF. The RBF keeps one fixed width everywhere, while the Gibbs kernel narrows where the length scale $\ell(x)$ is small and widens where it is large. Setting the variation to zero makes $\ell(x)$ go flat, so Gibbs collapses back onto the RBF. For Wall, $x$ is sequence position and $\ell(x_0) = 1/\sqrt{F}$ is set by the accumulated forget gate.

Wall Attention is a Gibbs kernel

Wall scores with a per-channel cumulative forget gate $F_{ij,n} = \prod_{r=j+1}^{i} g_{r,n} \in (0,1]$, giving $s_{ij} = \sum_n F_{ij,n}\, q_{i,n}\, k_{j,n}$ [1]. If we apply the identity $qk = \tfrac12(q^2 + k^2 - (q-k)^2)$ channel by channel, the score splits into an amplitude and a squared distance.

$$s_{ij} = \underbrace{\tfrac12 \sum_n F_{ij,n}\left(q_{i,n}^2 + k_{j,n}^2\right)}_{\log A_{ij}} \;-\; \sum_n \frac{(q_{i,n} - k_{j,n})^2}{2\,\ell_{ij,n}^2}, \qquad \boxed{\;\ell_{ij,n}^2 = \frac{1}{F_{ij,n}} = \prod_{r=j+1}^{i} g_{r,n}^{-1}\;}$$

Wall's kernel is a squared-exponential with a per-channel length scale $\ell_{ij,n}$ and an amplitude $A_{ij}$ out front, which is exactly the Gibbs kernel.

$$\kappa^{\mathrm{Wall}}(q_i, k_j) = A_{ij} \cdot \exp\!\left(-\sum_n \frac{(q_{i,n} - k_{j,n})^2}{2\,\ell_{ij,n}^2}\right)$$

The length scale $\ell_{ij,n}^2 = 1/F_{ij,n}$ grows with the accumulated forgetting between the two positions. One perspective on per-channel forgetting is just as per-channel length-scale growth. Intuitively, a channel whose gate stays open ($g_{r,n} \approx 1$) keeps $\ell_{ij,n} \approx 1$ and acts like vanilla NoPE attention while a channel that closes inflates its length scale as the gap widens, flattening its contribution until it drops out. The fixed Gibbs hyperparameter $\ell = 1$ becomes a learned, data-dependent length-scale field.

Where the Gibbs kernel comes from

I really enjoy reading classical statistical learning theory, and the Gibbs kernel is a beautiful example. To give some context, it comes out of Gibbs's thesis on adaptive Bayesian models [6] and sits inside the standard treatment of non-stationary covariance functions for Gaussian process regression [7]. It shows up wherever you want kernel regression with smoothness that varies across the input, for example in geostatistics and spatial models. Wall Attention is the same construction, specialized to attention and learned end to end.

One small caveat is that the Gibbs uses the symmetric combination $\ell_n(x)^2 + \ell_n(x')^2$, which is what keeps the kernel positive-definite, and Wall's length scale $\ell_{ij,n}^2 = \exp(P_{j,n} - P_{i,n})$ is directed from query to key over the span $(j, i]$. In a sense, Wall is the causal specialization of the Gibbs kernel.

Diagonal gates (and their generalization to Wall) have strong empirical results but always felt unsatisfying from a theoretical perspective. While scalar weight decay for RNNs arises from normal L2 regularization and weight decay, diagonal gates impose a per-channel geometry that does not have clear precedent. With the Gibbs kernel, we can see that Wall Attention is a natural, principled extension of the RBF kernel in Nadaraya-Watson for better length generalization. We can rely on rich, well-established tools to understand Wall from the kernel regression perspective [7][8]. I think this is a good direction to go in to understand Wall Attention, and the future of nonparametric attention mechanisms, better.
