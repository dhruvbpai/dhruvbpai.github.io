The Loss Curve Ouroboros

May 2025
Click and drag to rotate. Scroll to zoom.

Sometimes it's fun to ask ridiculous quesitions. For example, what if we take the loss curve from a training run, rescale it into a valid learning rate range, and use it as the schedule for the next run? Then do it again. And again. The loss curve eats its own tail (hence Ouroborous). Each generation's output becomes the next generation's input through LR scheduling, forming a bizarre dynamical system over the space of LR schedules.

Let's walk through what happens. We start with a standard warmup-stable-decay schedule. The model trains well, loss drops like a power law. Now we rescale that loss curve into an LR range. Since the loss decayed to near zero, the resulting schedule starts high but quickly falls to almost nothing. Training with that schedule barely moves the model. The loss stays high. But a high, flat loss curve becomes a high, flat LR schedule. So the next generation actually converges again. Then the cycle repeats. The system oscillates between "good schedule, good convergence" and "bad schedule, bad convergence" before (sometimes) settling into a fixed point.

Formalizing the problem

We need some notation. Suppose we have a model with parameters $\theta$ trained by gradient descent on a loss $\mathcal{L}$. Given a learning rate schedule $\eta : [0, T] \to \mathbb{R}_{>0}$, the training dynamics produce a loss curve $L : [0, T] \to \mathbb{R}_{\geq 0}$ defined by

$$L(t) = \mathcal{L}(\theta(t))$$

where $\theta(t)$ evolves according to

$$\dot{\theta}(t) = -\eta(t) \nabla \mathcal{L}(\theta(t))$$

The loss curve depends on the schedule, so we can think of it as a functional $\mathcal{F}$ that maps schedules to loss curves.

$$\mathcal{F} : \eta(\cdot) \mapsto L(\cdot)$$

Now we need the other half of the loop. The rescaling operator $\mathcal{R}$ takes a loss curve and produces a learning rate schedule by normalizing it into a target range $[\eta_{\min}, \eta_{\max}]$.

$$\mathcal{R}[L](t) = \eta_{\min} + (\eta_{\max} - \eta_{\min}) \cdot \frac{L(t) - L_{\min}}{L_{\max} - L_{\min}}$$

where $L_{\min} = \min_t L(t)$ and $L_{\max} = \max_t L(t)$. High loss maps to high learning rate, low loss maps to low learning rate.

The ouroboros is the composition $\mathcal{R} \circ \mathcal{F}$. Starting from some seed schedule $\eta^{(0)}$, we iterate

$$\eta^{(n+1)} = \mathcal{R} \circ \mathcal{F}[\eta^{(n)}]$$

A fixed point of this iteration is a schedule $\eta^*$ satisfying $\eta^* = \mathcal{R} \circ \mathcal{F}[\eta^*]$. In other words, a schedule whose loss curve, when rescaled, reproduces itself exactly.

Notice something useful. The schedule enters the dynamics only through the integrated schedule $A(t) = \int_0^t \eta(\tau) \, d\tau$. This means $\dot{A}(t) = \eta(t)$, so at a fixed point we get an ODE

$$\dot{A}(t) = \mathcal{R}[\mathcal{F}_A(A(t))]$$

where $\mathcal{F}_A$ writes the loss as a function of the cumulative schedule $A$ rather than time directly. This is a first-order autonomous ODE in $A$, which is a much nicer object to study than a functional fixed-point equation.

Example 1: Linear autoencoder

Let's find a setting where $\mathcal{F}$ has a closed form. Consider data $x \sim \mathcal{N}(0, \Sigma)$ with covariance $\Sigma = U \operatorname{diag}(\lambda_1, \dots, \lambda_d) U^\top$ where $\lambda_1 \geq \lambda_2 \geq \cdots \geq \lambda_d > 0$. Our model is a linear autoencoder $\hat{x} = W^\top W x$ with $W \in \mathbb{R}^{k \times d}$, so the bottleneck has rank $k$. The MSE loss is

$$\mathcal{L}(W) = \frac{1}{2} \mathbb{E}\left[\|x - W^\top W x\|^2\right]$$

This is the setting analyzed by Baldi & Hornik (1989) and Saxe et al. (2014). The key insight is that in the eigenbasis of $\Sigma$, the dynamics decouple. Each eigenmode $i$ has a projection strength $p_i(t) \in [0, 1]$ measuring how much of eigendirection $i$ the network has captured. The loss decomposes as

$$L(t) = \frac{1}{2} \sum_{i=1}^{d} \lambda_i \left(1 - p_i(t)\right)^2$$

Under gradient flow with schedule $\eta(t)$, each mode evolves independently according to a logistic equation

$$\dot{p}_i = 2\eta(t) \, \lambda_i \, p_i(1 - p_i)$$

This has an exact solution. Since the right-hand side factors as $\eta(t)$ times a function of $p_i$ alone, the schedule enters only through the integrated schedule $A(t) = \int_0^t \eta(\tau) \, d\tau$ just as we hoped.

$$p_i(t) = \sigma\!\left(2\lambda_i A(t) + \operatorname{logit}(p_i(0))\right)$$

where $\sigma$ is the sigmoid and $\operatorname{logit}(p) = \log(p/(1-p))$. Each mode traces out a sigmoid in the cumulative schedule $A$, with rate proportional to its eigenvalue $\lambda_i$. Big eigenvalues get learned first. Small ones take exponentially longer.

Substituting back, we get the closed-form training operator

$$\mathcal{F}[\eta](t) = \frac{1}{2} \sum_{i=1}^{d} \lambda_i \left(1 - \sigma\!\left(2\lambda_i A(t) + c_i\right)\right)^2$$

where $c_i = \operatorname{logit}(p_i(0))$ and $A(t) = \int_0^t \eta(\tau) \, d\tau$. This is $\mathcal{F}$ written explicitly as a function of the schedule. It is a sum of squared sigmoid tails, each indexed by an eigenvalue.

The ouroboros iteration $\eta^{(n+1)} = \mathcal{R} \circ \mathcal{F}[\eta^{(n)}]$ is now fully computable. And the fixed-point ODE becomes

$$\dot{A} = \eta_{\min} + (\eta_{\max} - \eta_{\min}) \cdot \frac{g(A) - g_{\min}}{g_{\max} - g_{\min}}$$

where $g(A) = \frac{1}{2} \sum_i \lambda_i (1 - \sigma(2\lambda_i A + c_i))^2$ is the loss as a function of the cumulative schedule. This is a separable first-order ODE. The right-hand side is a known function of $A$ alone, so we can solve it (at least numerically) and study its fixed points directly.
